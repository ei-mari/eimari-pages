## Publish with GitHub Desktop

1. Open GitHub Desktop.
2. Choose `Add an Existing Repository...`.
3. Select `/Users/eight/Documents/EIMARI`.
4. Review changed files and commit to `main`.
5. Click `Publish repository`.
6. On GitHub, open repository `Settings > Pages`.
7. Under `Build and deployment`, choose `Deploy from a branch`.
8. Select branch `main` and folder `/root`.
9. Save and wait a few minutes.

## Expected URLs

- `/` -> main LP
- `/onicamp.html` -> Oni Camp page
- `/junko-offer.html` -> Junko special offer page
